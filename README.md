# Contacts-Api

## Installation
1. Clone this repository into your local machine:
```
git clone https://github.com/nzediegwu1/contacts-api.git
```
2. Install dependencies
```
yarn
```
3. Start the development application by running
```
yarn dev
```
4. To run tests
```
yarn test
```
